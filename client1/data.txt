Файл, директория,symlink - ссылка на существующий файл,named pipe - для обмена данными между процессами,device file - драйвер, socket - как named pipe, но поддерживает множественные подключения. Настроить права можно файлам,директориям,symlink. Возможность настраивать права именно этим файлам дана, т.к. они могут быть созданы пользователем.
Тип - директория, права пользователя - rwx,группы - rx, всех - rx, владелец - allexserv, группа - nogroup, размер - 4096, дата создания - 2009-05-31 14:58, имя - phpBB3.
Изменять файл /etc/group разрешено только root. Можно разрешить управлять группами рядовым пользователям если изменять этот файл от root.
Может, если у A будут права wx на его каталог. Если каталог пользователя B будет пустым, то A должен иметь право r на каталог B, а если не пустой, то права wx на каталог B и все дочерние подкаталоги каталога B.
sudo groupmod -n dogs cats
